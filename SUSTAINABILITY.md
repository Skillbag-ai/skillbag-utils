# SkillBag Utils Sustainability

SkillBag Utils supports the SkillBag ecosystem by providing reusable skills for
common workspace maintenance workflows.

## Funding Goals

Funding for this repository should support:

- maintenance of existing utility skills
- tests and fixtures for skill behavior
- documentation and examples
- security review of install and adaptation workflows
- compatibility updates as the core standard evolves
- development of reference utilities such as validators or conformance helpers

## Relationship to Core Funding

The core standard should remain freely implementable. Funding this repository
supports practical tooling around the standard, not exclusive access to the
standard itself.

If formal sponsorship or membership is created, it should be coordinated with
the core SkillBag sustainability model.

## Commercial Work

Commercial services may be built around these utilities, including:

- custom skill creation
- migration from agent-specific skill formats
- workspace audits
- enterprise policy and conformance preparation

Commercial services should not imply that utility behavior is normative unless
that behavior is also defined by the core `SKILLBAG.md` specification.
